DATA RETRIEVAL SYSTEMS AND INTERNATIONAL PUBLISHING.
     
DESCRIBES PROBLEM AREAS IN THE ECONOMICS OF DATA BASE VENDING, BIBLIOGRAPHIC
DATA BASE PUBLISHING AND PRIMARY JOURNAL ARTICLE PUBLISHING. DISCUSSES DATA
BASE HOSTS; DEVELOPMENTS IN THE DATA BASE PUBLISHING INDUSTRY; LINKING TEXTS
AND GRAPHICS; INTEGRATED DATA BASE PUBLISHING; AND RESEARCH IN DOCUMENT
DELIVERY, INCLUDING A DESCRIPTION OF THE ADONIS CONCEPT. CONCLUDES THAT A MUCH
CLOSER RELATIONSHIP NEEDS TO BE FORGED BETWEEN DATA BASE PUBLISHERS,
LIBRARIANS AND END USERS TO ALERT, AT THE EARLIEST POSSIBLE STAGE, ANY
MISMATCH BETWEEN APPLICATION OF TECHNOLOGY AND USER ACCEPTABILITY. ALSO
STRESSES THE NEED FOR USEFUL AND APPROPRIATE STANDARDS.
