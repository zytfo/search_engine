NETWORK FILE STORAGE SYSTEM.
     
THE COMMON FILE SYSTEM (CFS) IS A LARGE ON-LINE CENTRALISED STORAGE SYSTEM FOR
THE LOS ALAMOS NATIONAL LABORATORY'S COMPUTER NETWORK. THE CFS PROVIDES LOS
ALAMOS COMPUTER USERS A RELATIVELY SIMPLE SET OF PRIMITIVES WITH WHICH THEY
CAN STORE AND RETRIEVE FILES. A TREE-STRUCTURED DIRECTORY ALLOWS THE USERS TO
ORGANISE THEIR DATA IN A LOGICAL AND REASONABLE MANNER.
