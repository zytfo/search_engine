A PERSONAL, COMPUTERISED LITERATURE RETRIEVAL SYSTEM.

THE COMPUTERISED METHOD OF INDEXING AND RETRIEVING DOCUMENTS FROM A PERSONAL
COLLECTION IS DESCRIBED. CONSISTS OF A UNIVAC 1108 WITH INPUT FROM CARDS AND
OUTPUT VIA LINE PRINTER. SYSTEM ALLOWS INPUT OF AUTHOR, TITLE, BIBLIOGRAPHIC
DETAILS AND KEYWORDS WHILE DOCUMENTS ARE ALLOCATED A NUMBERIC CODE AND STORED
SEQUENTIALLY. 3 TYPES OF OUTPUT ARE PROVIDED; ALPHABETIC KEYWORDS LIST WITH
DOCUMENT REFERENCE NUMBERS, A REFERENCE LIST OF DOCUMENTS TITLES,
BIBLIOGRAPHIC DETAILS AND AUTHORS, AND OUTPUT FROM SEARCHES ON THE SYSTEM.
