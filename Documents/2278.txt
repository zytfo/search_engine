IMPLEMENTATION OF PROCESS MANAGEMENT FOR A SECURE ARCHIVAL STORAGE SYSTEM.
     
THIS THESIS PRESENTS AN IMPLEMENTATION OF PROCESS MANAGEMENT FOR A SECURITY
KERNEL BASED SECURE ARCHIVAL STORAGE SYSTEM (SASS). THE IMPLEMENTATION IS
BASED ON A FAMILY OF SECURE, DISTRIBUTED, MULTI-MICROPROCESSOR OPERATING
SYSTEMS DESIGNED TO PROVIDE MULTILEVEL INTERNAL SECURITY AND CONTROLLED
SHARING OF DATA AMONG AUTHORISED USERS.
