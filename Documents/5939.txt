REFERENCE RETRIEVAL WITHOUT USER QUERY FORMULATION.

DESCRIBES A COMPUTER PROGRAM FOR THE PURPOSE OF SATISFYING INCOMPLETELY
DEFINED USER NEEDS THROUGH A MAN-MACHINE DIALOGUE WHICH DOES NOT REQUIRE A
QUERY FORMULATION BY THE USER. THE PROGRAM ATTEMPTS TO SATISFY THE USER'S
NEEDS FROM A MODEL WHICH IT BUILDS OF HIS INTERESTS; A MODEL WHICH IS A SUBSET
OF ITS OWN WORLD MODEL (THE DATA BASE). EXPLORES THE POSSIBILITIES OF THE
PROGRAM CREATING VIRTUAL WORLD MODELS TO PREVENT THE NEED FOR UNNECESSARILY
ACCESSING THE WHOLE DATA BASE AND INCORPORATING A FORM OF WEIGHTING BASED ON
ASSOCIATION STRENGTHS BETWEEN ITEMS IN THE DATA BASE.
