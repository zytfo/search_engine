LITERATURE STORAGE AND RETRIEVAL SYSTEM USING A TABLE-TOP COMPUTER.
     
A DIALOGUE-PROGRAM IS DESCRIBED WHICH HANDLES CROSS-REFERENCES WITHIN A GIVEN
SET OF KEYWORDS, AUTHOR NAMES, JOURNAL NAMES AND SPECIFIED REFERENCE NUMBERS.
IT USES THE SET OF BASIC STATEMENTS AVAILABLE IN A WANG 2200T VERSION WHICH IS
CONNECTED TO A FLOPPY-DISKETTE. THE PROGRAM HAS CONVENIENT FEATURES FOR
EDITING AND PROCESSING OF LITERATURE DATA.
