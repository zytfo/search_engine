THE COMPUTER AS AN INSTRUCTIONAL DEVICE; NEW DIRECTIONS FOR LIBRARY USER
EDUCATION.
     
CONTRIBUTION TO A THEMATIC ISSUE ON LIBRARY USE INSTRUCTION. DISCUSSES THE
IMPLICATIONS OF AUTOMATION FOR LIBRARY USER EDUCATION, AND ARGUES THAT THE USE
OF COMPUTERS WILL BRING ABOUT A TOTAL REDEFINITION OF THE ROLE AND FUNCTION OF
LIBRARY USER EDUCATION. CONSIDERS A NUMBER OF NEW AREAS WITH WHICH USER
EDUCATION LIBRARIANS WILL NEED TO ACQUAINT THEMSELVES IN ORDER TO ACTIVELY
PARTICIPATE IN THESE DEVELOPMENTS.
