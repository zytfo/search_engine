ENVIRONMENTAL ASSESSMENT DATA SYSTEMS USER GUIDE' PROJECT PROFILE SYSTEM.
     
THE REPORT IS A USER GUIDE TO THE PROJECT PROFILE SYSTEM (PPS), A COMPUTERISED
DATA BASE ON PROJECTS OF VARIOUS TYPES THAT PERTAIN TO SPECIFIC INDUSTRIAL
CATEGORIES. PPS IS 1 OF 5 DATA BASES WHICH ARE COMPONENTS OF THE ENVIRONMENTAL
ASSESSMENT DATA SYSTEMS (EADS), DESIGNED TO AID RESEARCHERS IN ENVIRONMENTAL
ASSESSMENT, EMISSIONS CHARACTERISATION, AND CONTROL TECHNOLOGY DEVELOPMENT.
