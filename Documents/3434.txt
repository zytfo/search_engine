DEVELOPMENT OF AN APPLICATION SOFTWARE FOR ON-LINE INFORMATION RETRIEVAL.
     
A SOFTWARE PACKAGE HAS BEEN DEVELOPED FOR ON-LINE SEARCHING THE INSPEC DATA
BASE. CARE HAS BEEN TAKEN TO MAINTAIN THE PORTABILITY OF THE PACKAGE SO THAT
IT CAN BE RUN ON ANY THIRD GENERATION COMPUTER SYSTEM. DETAILS OF THE PACKAGE
AND ITS UTILISATION FOR SDI SERVICE HAVE BEEN DESCRIBED.
