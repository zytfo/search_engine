A DESCRIPTION OF THE BROADCAST TELIDON SYSTEM.
     
THE DATA FORMAT WHICH HAS BEEN ADOPTED FOR THE TELIDON SYSTEM IS DESCRIBED.
THE SYSTEM REQUIREMENTS ARE LISTED FOLLOWED BY A DETAILED ACCOUNT OF THE
STRUCTURE USED TO TRANSMIT DATA SUCH THAT THESE REQUIREMENTS ARE MET. THE
ALPHA-GEOMETRIC CODING ALLOWS DISPLAY TECHNOLOGY AND RESOLUTION TO BE
INDEPENDENT OF THE CODES.
