LIBRARY DATA COLLECTION HANDBOOK.
     
DESCRIBES BASIC MANAGEMENT INFORMATION USEFUL IN ACADEMIC, PUBLIC, SCHOOL, AND
SPECIAL LIBRARIES. AN UNDERLYING PREMISE OF THE HANDBOOK IS THAT ALL TYPES OF
LIBRARIES HAVE A COMMON SET OF FUNCTIONS, PURPOSES, AND RESOURCES WHICH
OUTWEIGH THE DIFFERENCES IN SETTING, SIZE, OR ORGANISATIONAL GOALS. PRESENTS A
SET OF CATEGORIES OF INFORMATION THAT ARE JUDGED TO BE IMPORTANT IN DESCRIBING
THE LIBRARY, AND PROVIDES DEFINITION OF THOSE CATEGORIES.
