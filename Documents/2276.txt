COEXISTENCE OF GRAPH-ORIENTED AND RELATIONAL DATA FILE ORGANISATIONS IN A DATA
BANK.
     
IT IS SHOWN THAT A COEXISTENCE OF HIERARCHICAL AND RELATIONAL DATA BANK
STRUCTURES IN COMPUTER NETWORKS IN A COMMON DATA BANK SYSTEM IS POSSIBLE. THE
COEXISTENCE MODEL REGARDS THE GRAPH THEORY CODASYL APPROACH AND CODD'S
RELATIONAL MODEL AS GRAPH-ORIENTED, OR RATHER TABLE-ORIENTED, DATA FILE
ORGANISATION AS PRESENTED TO THE USER OF A COMMON LOGICAL STRUCTURE OF THE
DATA BANK.
